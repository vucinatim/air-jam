# For Agents

This page is the fastest entrypoint for LLM agents and automation tools.

## What Air Jam Is

Air Jam is a multiplayer game platform and SDK where players join sessions by scanning a QR code on their phone.  
Hosts run in your game, inputs are routed through the Air Jam server, and your game consumes typed controller state via React hooks.
