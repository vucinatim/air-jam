# Security Policy

## Supported Versions

Security fixes are prioritized for:

1. The `main` branch
2. The latest published npm versions of:
   - `@air-jam/sdk`
   - `@air-jam/server`
   - `create-airjam`

Older versions may receive fixes at maintainer discretion.

## Reporting a Vulnerability

Please report vulnerabilities privately.

1. Email: `tim.vucina@gmail.com`
2. Include:
   - affected package/app and version
   - impact description
   - reproduction steps or proof of concept
   - any suggested mitigation

Please do not open public GitHub issues for security vulnerabilities.

## Response Expectations

1. Initial acknowledgment target: within 72 hours.
2. Triage and severity assessment after acknowledgment.
3. Fix and coordinated disclosure timeline depends on severity and exploitability.
